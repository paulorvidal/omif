🧮 Plataforma Nacional OMIF (Olimpíada de Matemática das Instituições Federais)
💻 Sobre o Projeto

A Plataforma Nacional da OMIF é um sistema web desenvolvido como projeto de extensão no Instituto Federal do Paraná (IFPR). O objetivo principal da aplicação é digitalizar, organizar e gerenciar a comunicação e a aplicação da Olimpíada de Matemática das Instituições Federais, conectando estudantes e professores de toda a Rede Federal do Brasil.

O projeto foi construído com foco em escalabilidade, alta disponibilidade e acessibilidade, garantindo uma experiência de usuário (UX) fluida para milhares de acessos simultâneos durante os períodos de avaliação, além da aplicação de ferramentas de inclusão para melhorar a experiência de todos os usuários.
🚀 Minhas Contribuições (Front-end Developer)

Atuei no desenvolvimento da interface do sistema, integrando uma equipe multidisciplinar de 7 desenvolvedores. Minhas principais responsabilidades incluíram:

    Desenvolvimento de Interfaces: Implementação de 7 interfaces dinâmicas e responsivas utilizando React.js e TypeScript, garantindo usabilidade em diferentes dispositivos.

    Integração de APIs: Colaboração ativa na definição dos contratos de API REST (Swagger/OpenAPI) para assegurar uma comunicação eficiente, segura e bem documentada com o Back-end.

    Boas Práticas & Arquitetura: Auxílio na modelagem e estruturação da lógica de negócio no front-end, focando na separação de responsabilidades e na manutenção do código limpo (Clean Code).

    Trabalho em Equipe e Versionamento: Utilização do padrão Gitflow para o controle de ramificações, além da participação rigorosa em rotinas de Code Review para garantir a estabilidade das entregas críticas do projeto.

🛠️ Tecnologias e Ferramentas Utilizadas

    Front-end: React.js, TypeScript, JavaScript (ES6+), HTML5, CSS3.

    Integração: RESTful APIs, JSON, Contratos de API.

    Versionamento & Metodologia: Git, GitHub, Gitflow, Code Review, Metodologias Ágeis.
